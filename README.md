# LocalBasket 🛒

## 📌 About
LocalBasket ek e-commerce web app hai jahan users nearby stores se products order kar sakte hain.

## 🚀 Features
- Add to cart
- Order system
- Store management
- Single-store cart logic

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express
- MySQL
